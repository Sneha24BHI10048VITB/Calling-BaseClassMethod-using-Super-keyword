**Project Title**

Calling Base Class Method using super Keyword in Java

**Description**

This project demonstrates how the super keyword is used in Java to call a method from the parent class after method overriding. The program uses a real‑world example of an Employee and Manager to make the concept easier to understand.

**Concepts Used**

Inheritance 

Method Overriding

super Keyword

Constructors

Object Oriented Programming

**Technologies Used**

Java Programming Language

JDK 8 or higher

Any Java IDE (Eclipse / IntelliJ / NetBeans / VS Co

**How to Run the Program**

Open any Java IDE

Create a new Java project

Create a new Java file named SuperMethodExample.java

Copy the program code into the file

Compile the program

Run the program
